# 介绍
你应该已经注册完 Milk SMP Server 的 游戏帐户 了。接下来，或许该介绍下服务器了。

服务器核心采用 [Leaves 1.21.3](https://leavesmc.org/software/leaves) 驱动，这使得建造红石机器变得异常容易。
服务器内有经济系统、小游戏等，支持假人，丰富玩家游玩体验。
